React-приложение "Образовательный портал", на TypeScript, архитектура на FSD.
Участник читает статью и во время чтения, проходит промежуточные тесты.

/** Структура статьи */
export interface Article {
  id          : string
  title       : string
  description : string
  blocks      : ContentBlockType[]
  finalTest   : TestQuestionType[]
  coverImage? : string
  duration    : number // в минутах
  difficulty  : Difficulty
  tags        : string[]
  createdAt   : string
}

Статья состоит из блоков контента.

/** Тип блока контента */
export interface ContentBlockType {
  id         : string
  type       : 'theory' | 'test'
  content    : string // для теории
  questions? : TestQuestionType[] // для тестовых блоков
}

/** Тип теста */
export interface TestQuestionType {
  id            : string
  text          : string
  options       : string[]
  correctAnswer : number
}

/** Ответы пользователя */
export type TestUserAnswers = Record<string, number>

TrueFalse — верно/неверно с объяснением
OrderSteps — расставить шаги в правильном порядке
